# AI Interview Scheduler 🤖📞

An enterprise-grade Voice AI application that automates interview scheduling using Maya, an AI voice agent built on Bolna.

## Problem
HR teams waste hours calling candidates manually to schedule interviews.

## Solution
Maya (Voice AI Agent) automatically calls candidates, checks availability, and schedules interviews — without any human involvement.

## Flow
User fills form → Web app triggers Maya → Maya calls candidate → Candidate confirms slot → Dashboard updates in real time

## Tech Stack
- **Frontend**: React + Vite
- **Backend**: Node.js + Express
- **Voice AI**: Bolna
- **LLM**: GPT-4.1 Mini
- **Voice**: ElevenLabs
- **Telephony**: Plivo
- **Tunnel**: ngrok

## Setup
### Backend
cd interview-scheduler-backend
npm install
npm run dev

### Frontend
cd interview-scheduler-frontend
npm install
npm run dev
